# Hello, Самурай!

Это полностью подготовленный проект для "Пути самурая", где установлены ВСЕ библиотеки нужных версий.
В процессе прохождения видео-уроков не нужно устанавливать пакеты. Просто проверяем, что библиотека есть в package.json.
В случае, если ее не находим или замечаем неправильную ее работу (отличную от Диминой версии),
пишем об этом в Telegram `Валере Сафронову (@safronman)` или `Марго (@margokomilfo)`.

В этом стартовом проекте установлены следующие библиотеки

- **antd**
- **axios**
- **classnames**
- **formik**
- **gh-pages**
- **jest**
- **react** and **@types/react**
- **react-dom** and **@types/react-dom**
- **react-redux** and **@types/react-redux**
- **react-router-dom** and **@types/react-router-dom**
- **react-scripts**
- **react-test-renderer** and **@types/react-test-renderer**
- **redux**
- **redux-form** and **@types/redux-form**
- **redux-thunk**
- **reselect**
- **typescript**
- **uuid** and **@types/uuid**

После git clone 'link' установите все зависимости из package.json:

### `yarn`

а потом запустите проект:

### `yarn start`
